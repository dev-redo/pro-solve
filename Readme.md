## 🖐 프로솔브 소개하기

안녕하세요.

**프로솔브** 는 유저가 제출한 모든 풀이를 저장한 후 보여주는 것을 목표로 합니다.

현재 프로그래머스 사이트는 처음으로 성공한 각 언어의 풀이만을 유저에게 보여주고 있습니다.

코딩 테스트 스터디를 참가하면서 이 점을 아쉬워 하는 유저들이 많다는 것을 발견하고 모든 풀이를 저장하는 익스텐션을 개발하게 되었습니다.
